# Career Roadmap Templates — 30/60/90/120/180/360 Day Plans
**Step-by-step action plans. No vague advice. Every day counts.**

## HOW THE ROADMAP ENGINE WORKS
Based on candidate's gap analysis score:
- **Score 80-100:** You're ready. 30-day sprint to first interview.
- **Score 60-79:** Almost ready. 90-day plan — fix gaps, then apply.
- **Score 40-59:** Significant gaps. 180-day plan — upskill first.
- **Score 0-39:** Major rebuild. 360-day transformation.

## 120-DAY PLAN (Score 50-59: Career Pivot Needed)

### Month 1: Discover & Plan
- Run GovRAG Decision Engine — get full gap analysis + career pivots
- Choose pivot direction (highest transferability from GovRAG suggestion)
- Research target role: salary, demand, required skills, certs
- Find 3 people already in target role — ask for advice

### Month 2: Learn & Build
- Enroll in certification program for target role
- Start building portfolio project in new field
- Join communities in target industry
- Rebrand LinkedIn for new direction

### Month 3: Transition
- Complete certification
- Finish portfolio project with measurable results
- Update resume for new role (highlight transferable skills)
- Start applying (target companies that value career changers)

### Month 4: Execute
- Apply to 20+ positions in new field
- Network specifically in new industry
- Accept transitional role if needed (stepping stone)
- Continue learning on the job

**120-Day Target:** New certification, portfolio, 20+ applications in new field, 1 offer.

## 360-DAY PLAN (Score 0-39: Complete Career Rebuild)

### Quarter 1 (Months 1-3): Foundation
- **Week 1:** Run GovRAG. Accept reality. Set 1-year goal.
- **Month 1:** Choose new career direction. Research everything.
- **Month 2:** Start online degree/diploma/bootcamp.
- **Month 3:** Learn fundamentals. Build first small project.

### Quarter 2 (Months 4-6): Skills Building
- Complete core training program
- Get first certification
- Build 3 portfolio projects
- Start freelancing/volunteering for experience

### Quarter 3 (Months 7-9): Experience Building
- Get internship, contract, or entry-level position
- Continue learning advanced skills
- Get second certification
- Build professional network (100+ connections)

### Quarter 4 (Months 10-12): Career Launch
- Upgrade resume with real experience + projects + certs
- Apply to target-level positions
- Negotiate from position of strength
- Achieve career transformation

**360-Day Target:** New career, 2+ certs, portfolio, real experience, competitive salary.

## ROADMAP RULES (built into Decision Engine)

```python
def generate_roadmap(score, gaps, level, target):
    if score >= 80:
        return "30-day sprint"   # Ready to compete
    elif score >= 70:
        return "60-day plan"     # Minor gaps
    elif score >= 60:
        return "90-day plan"     # Significant gaps
    elif score >= 50:
        return "120-day plan"    # Career pivot
    elif score >= 40:
        return "180-day plan"    # Major transformation
    else:
        return "360-day plan"    # Complete rebuild
```
