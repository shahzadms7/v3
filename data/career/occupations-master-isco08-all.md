## HOW TO USE THIS DATABASE

Each occupation entry contains:
- **ISCO Code:** 4-digit unit group
- **Title:** Official ISCO-08 name + common aliases
- **Tasks:** 5-8 core work activities (what the job actually does)
- **Hard Skills:** Technical competencies required
- **Soft Skills:** Behavioral competencies required
- **Tools/Software:** Platforms commonly used
- **Qualifications:** Typical education/certs needed
- **Salary Range:** Global USD ranges (entry → senior)
- **Demand 2026:** Growth outlook
- **AI Risk:** Automation displacement risk
- **Similar Roles:** ISCO codes of adjacent occupations
- **JD Template:** Standard job description template
